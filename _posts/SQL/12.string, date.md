# 문제
<https://school.programmers.co.kr/learn/courses/30/lessons/59047?language=oracle>

## 테이블 구조

1. `ANIMAL_INS`테이블
    - `ANIMAL_ID`: 동물의 아이디
    - `ANIMAL_TYPE`: 생물 종
    - `DATETIME`: 보호 시작일
    - `INTAKE_CONDITION`: 보호 시작 시 상태
    - `NAME`: 이름
    - `SEX_UPON_INTAKE`: 성별 및 중성화 여부

## 답안


```sql
SELECT ANIMAL_ID, NAME
FROM ANIMAL_INS
WHERE UPPER(NAME) LIKE '%EL%' AND ANIMAL_TYPE = 'Dog'
ORDER BY NAME
```

## 쿼리 설명


```sql
WHERE UPPER(NAME) LIKE '%EL%' AND ANIMAL_TYPE = 'Dog'
```

- Oracle에서는 대소문자를 구별하여 조건이 정확하게 일치하는 데이터만 검색한다.
- `UPPER(NAME)`: NAME을 전부 대문자로 변환
- `LIKE '%EL%'`: 문자열 안에 `EL`이 어디든 포함되면 매칭
