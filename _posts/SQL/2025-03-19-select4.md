## 테이블 구조

1. `FOOD_FACTORY`테이블
    - `FACTORY_ID`: 공장 ID
    - `FATORY_NAME`: 공장 이름
    - `ADDRESS`: 주소
    - `TLNO`: 전화번호

## 답안


```sql
SELECT FACTORY_ID, FACTORY_NAME, ADDRESS
FROM FOOD_FACTORY
WHERE ADDRESS LIKE '강원도%'
ORDER BY FACTORY_ID;
```

## 쿼리 설명


```sql
SELECT FACTORY_ID, FACTORY_NAME, ADDRESS
```

- `FOOD_FACTORY`테이블에서 데이터를 가져온다.
