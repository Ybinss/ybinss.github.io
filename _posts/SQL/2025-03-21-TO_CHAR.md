# `TO_CHAR`함수 정리
`TO_CHAR`함수는 날짜(DateTime) 데이터를 원하는 형식으로 변환할 때 사용한다.

## I. 기본 문법


```python
TO_CHAR(날짜_컬럼, '형식')
```

- `날짜_컬럼`: 변환할 날짜 데이터
- `'형식'`: 출력할 날짜 및 시간 형식

## II. 날짜 포맷 코드

|형식 코드|설명|예제(2024-03-15 14:30:45)|
|:---|:---|:---|
|`YYYY`|연도(4자리)|2024|
|`YY`|연도(2자리)|24|
|`MM`|월(2자리)|03|
|`MM` or `FMMM`|월(1~12)|3|
|`DD`|일(2자리)|15|
|`D` or `FMDAY`|일(1~31)|15|
|`DAY`|요일(전체)|Friday|
|`DY`|요일(약어)|Fri|
|`HH24`|시간(24시간제, 2자리)|14|
|`HH`|시간(12시간제, 2자리)|02|
|`MI`|분(2자리)|30|
|`SS`|초(2자리)|45|
|`AM`/`PM`|AM/PM|PM|

## III. 사용 예제

(1) 날짜를 `YYYY-MM-DD`형식으로 변환


```python
SELECT TO_CHAR('2024-03-15 14:30:45', 'YYYY-MM-DD') AS formatted_date
FROM DUAL;
-- 결과: 2024-03-15
```

(2) 날짜를 `YYYY년 MM월 DD일`형식으로 변환


```python
SELECT TO_CHAR(TO_DATE('2024-03-15', 'YYYY-MM-DD'), 'YYYY"년" MM"월" DD"일"') AS formatted_date
FROM DUAL;
-- 결과: 2024년 03월 15일
```

(3) 날짜와 시간을 `YYYY/MM/DD HH:MM:SS`형식으로 변환


```python
SELECT TO_CHAR('2024-03-15 14:30:45', 'YYYY-MM-DD HH24:MI:SS') AS formatted_datetime
FROM DUAL;
-- 결과: 2024/03/15 14:30:45
```

(4) 요일을 포함한 날짜 출력


```python
SELECT TO_CHAR(TO_DATE('2024-03-15', 'YYYY-MM-DD'), 'YYYY-MM-DD ("DAY:" DAY)') AS formatted_date
FROM DUAL;
-- 결과: 2024-03-15 (Friday)
```

(5) 12시간제(AM/PM)시간 표시


```python
SELECT TO_CHAR('2024-03-15 14:30:45', 'YYYY-MM-DD'), 'YYYY-MM-DD HH:MI:SS PM') AS formatted_time
FROM DUAL;
-- 결과: 2024-03-15 02:30:45 PM
```
