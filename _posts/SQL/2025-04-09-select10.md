# 문제
<https://school.programmers.co.kr/learn/courses/30/lessons/131535>

## 테이블 구조

1. `ANIMAL_INS`테이블
    - `USER_ID`: 회원 ID
    - `GENDER`: 성별
    - `AGE`: 나이
    - `JOINED`: 가입일

## 답안


```sql
SELECT COUNT(USER_ID) as USERS
FROM USER_INFO
WHERE YEAR(JOINED) = 2021 AND AGE >= 20 AND AGE <= 29;
```

## 쿼리 설명


```sql
SELECT COUNT(USER_ID) as USERS
```

- `COUNT(column)`: NULL을 제외한 행 개수를 셈
