## 테이블 구조

1. `ANIMAL_INS`테이블
    - `USER_ID`: 회원 ID
    - `GENDER`: 성별
    - `AGE`: 나이
    - `JOINED`: 가입일

## 1. 답안


```sql
SELECT COUNT(*) AS USERS
FROM USER_INFO
WHERE TO_CHAR(JOINED, 'YYYY') = 2021 AND AGE BETWEEN 20 AND 29
```

## 쿼리 설명


```sql
SELECT COUNT(*) AS USERS
```

- `COUNT(column)`: NULL을 제외한 행 개수를 셈
