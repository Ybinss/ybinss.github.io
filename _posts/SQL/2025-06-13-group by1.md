# 문제
<https://school.programmers.co.kr/learn/courses/30/lessons/132202>

## 테이블 구조

1. `APPOINTMENT`테이블
    - `APNT_YMD`: 진료예약일시
    - `APNT_NO`: 진료예약번호
    - `PT_NO`: 환자번호
    - `MCDP_CD`: 진료과코드
    - `MDDR_ID`: 의사ID
    - `APNT_CNCL_YN`: 예약취소여부
    - `APNT_CNCL_YMD`: 예약취소날짜

## 1. 답안(LIKE)


```sql
SELECT MCDP_CD AS "진료과코드", COUNT(*) AS "5월예약건수"
FROM APPOINTMENT
WHERE TO_CHAR(APNT_YMD, 'YYYY-MM') = '2022-05'
GROUP BY MCDP_CD
ORDER BY "5월예약건수", "진료과코드"
```

## 쿼리 설명


```sql
WHERE TO_CHAR(APNT_YMD, 'YYYY-MM') = '2022-05'
```

- `APNT_YMD`컬럼에서 2022년 5월인 행만 필터링한다.


```sql
GROUP BY MCDP_CD
```

- `MCDP_CD`기준으로 그룹화

## 주의 사항


```sql
SELECT MCDP_CD AS "진료과코드", COUNT(*) AS "5월예약건수"
ORDER BY "5월예약건수", "진료과코드"
```

- `ORDER BY`와 컬럼 별칭에 큰따옴표 사용
