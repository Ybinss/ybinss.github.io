# 문제
<https://school.programmers.co.kr/learn/courses/30/lessons/276035>

## 테이블 구조

1. `DEVELOPERS`테이블
    - `ID`: 개발자의 ID
    - `FIRST_NAME`: 이름
    - `LAST_NAME`: 성
    - `EMAIL`: 이메일
    - `SKILL_CODE`: 스킬 코드

## 1.답안(FROM 절 & 사용)


```sql
SELECT DISTINCT D.ID, D.EMAIL, D.FIRST_NAME, D.LAST_NAME
FROM DEVELOPERS D
JOIN SKILLCODES S ON S.CODE = D.SKILL_CODE & S.CODE
WHERE S.CATEGORY LIKE 'Front End'
ORDER BY ID
```

## 2.답안(WHERE 절 & 사용)


```sql
SELECT DISTINCT D.ID, D.EMAIL, D.FIRST_NAME, D.LAST_NAME
FROM DEVELOPERS D
JOIN SKILLCODES S
WHERE S.CODE = D.SKILL_CODE & S.CODE AND S.CATEGORY LIKE 'Front End'
ORDER BY ID
```
