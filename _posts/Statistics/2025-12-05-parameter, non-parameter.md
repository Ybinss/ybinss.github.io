# I. 모수적(parametric) 방법

> *데이터가 특정한 분포를 따른다고 가정하고, 그 분포를 몇 개의 모수(parameter)로 요약하는 방법이다.*

**< 핵심 아이디어 >**

- 데이터 전체의 모양을 **미리 정해진 분포 형태**로 가정
- 그 분포를 설명하는 **소수의 숫자(모수)**만 추정
- 대표적인 분포:
    + 정규분포, 이항분포, 포아송분포 등

**< 예시 >**

- "이 데이터는 정규분포를 따른다"
- 정규분포는 **평균($\mu$)**과 **분산($\sigma^2$)** 두 개의 모수로 결정됨
- 우리는 데이터로부터 $\mu$와 $\sigma^2$만 추정하면 됨

**< 대표적인 모수적 기법 >**

- t-test
- ANOVA
- 선형회귀(Linear Regression)
- 로지스틱 회귀
- 정규분포 기반 추정(MLE 등)

**< 장점 >**

- 데이터가 가정에 맞으면 **효율적이고 해석이 명확**
- 적은 데이터로도 안정적인 추정 가능
- 계산 비용이 낮음

**< 단점 >**

- **분포 가정이 틀리면 결과가 왜곡**
- 실제 데이터의 복잡한 형태를 잘 반영하지 못할 수 있음

# II. 모수적(non-parametric) 방법

> *데이터가 어떤 분포를 따른다는 가정을 거의 하지 않고, 데이터 자체의 구조를 그대로 활용하는 방법이다.*

**< 핵심 아이디어 >**

- 특정 분포 형태를 **가정하지 않음**
- 모수의 개수가 고정되어 있지 않거나, 아예 없음
- 데이터가 많을수록 모델이 더 유연해짐

**< 예시 >**

- "이 데이터가 정규분포인지 모르겠다"
- 분포를 가정하지 않고 **순위, 거리, 국소적 거리**를 사용
- 데이터가 늘어나면 표현력도 함께 증가

**< 대표적인 비모수적 기법 >**

- Wilcoxon rank-sum test
- Mann-Whitney U test
- Kruskal-Wallis test
- k-NN(k-최근접 이웃)
- 커널 밀도 추정(KDE)
- 결정트리

**< 장점 >**

- 분포 가정이 필요 없어 **현실 데이터에 강함**
- 이상치나 비대칭 분포에 비교적 안정적
- 복잡한 패턴을 잘 포착

**< 단점 >**

- 데이터가 적으면 성능이 불안정
- 해석이 직관적이지 않을 수 있음
- 계산 비용이 큰 경우가 많음

# III. 비유 설명

- **모수적 방법**
    + "데이터를 정규분포라는 틀에 맞춰 압축한다"
    + 전체를 대표하는 몇 개의 숫자만 본다 
- **비모수적 방법**
    + "틀을 정하지 않고 데이터 모양을 그대로 본다"
    + 데이터 하나하나의 상대적 위치와 관계를 활용한다

# IV. 비교 정리

|구분|모수적 방법|비모수적 방법|
|:---|:---|:---|
|분포 가정|있음|거의 없음|
|모수 개수|고정|고정되지 않음|
|데이터 요구량|적어도 가능|비교적 많이 필요|
|해석력|높음|상대적으로 낮음|
|유연성|낮음|높음|
|예시|t-test, 선형회귀|wilcoxon test, k-NN|
